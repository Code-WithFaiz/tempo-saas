# Tempo — SaaS Landing Page

A clean, modern landing page for a fictional AI productivity tool.
Light theme, indigo + lime accents, "pulse rhythm" signature visual.

## Features
- Animated pulse-bar visual in the hero dashboard mockup
- Feature grid, 3-step "how it works", pricing tiers
- Accordion FAQ section
- Fully responsive with mobile nav
- Respects `prefers-reduced-motion`

## Tech Stack
HTML5, CSS3 (custom properties), Vanilla JavaScript.

## Folder Structure
```
tempo-saas/
├── index.html
├── css/style.css
└── js/script.js
```

## How to customize
- **Product name/copy**: edit directly in `index.html` ("Tempo" is a
  placeholder concept — swap in a real product name and copy)
- **Pricing**: edit the price cards in the Pricing section of `index.html`
- **FAQ**: edit the `FAQS` array at the top of `js/script.js`
- **Logos strip**: replace the text `.logo-chip` divs with real client
  logo images once available

## Deploy
Push to GitHub, then enable GitHub Pages (same process as your other projects).
